# Lunasm

A assembly language writen in C++ for the [Luna16](https://github.com/Garoze/Console) fantasy console

## Installation

```
$ git clone --recursive https://github.com/Garoze/Lunasm
$ cd Lunasm
$ ./configure.sh 
$ ninja -C build
```

Run the project
```
$ bin/Lunasm
```
